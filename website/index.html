The agent problem
Your agents are getting smarter and running longer.
You have almost no idea what they're actually doing.

Modern agents — whether autonomous, multi-step, tool-using, or spawning sub-agents — produce complex chains of reasoning and actions. Without structured traces you lose cost control, can't debug failures, and have no audit trail.

Cost blindness
One runaway research loop can cost hundreds in API tokens before you notice.
Black box debugging
“Why did it call that tool 14 times?” — no way to replay the exact trajectory.
No accountability
For teams or regulated environments you need provenance, not just logs.
CAPABILITIES

Complete visibility.
Zero dependencies.

One lightweight tool that does the one thing that matters: tell you exactly what happened inside your agents.

One-line CLI wrap

Trace any existing agent instantly with zero code changes. Point the CLI wrapper at any command and it auto-detects and records tokens, tools, latency, and full execution graphs. The universal on-ramp for every agent.

Precise cost & token tracking

Built-in pricing for 15+ models. Per-agent, per-day, per-model breakdowns. Set budgets and get alerts before you blow past them.

Multi-agent trace trees

See the full hierarchy when agents spawn sub-agents. Parent/child linking, full execution graphs, and the ability to drill into any branch.

Beautiful local dashboard

Run agenttrace-io dashboard. Live stats, searchable runs, full trace drill-down, tool I/O, and export — all in your browser on localhost.

SDKs + framework middleware

Drop-in TypeScript and Python SDKs for custom code. Optional thin middleware for LangGraph, CrewAI and others. The CLI wrapper gives you full auto-detection for everything else — no framework required.

SQLite. Export. Escape hatches.

Everything lives in one agenttrace.db file. Export JSON/CSV. Forward to OpenTelemetry. Webhooks for Slack/Discord alerts. Full data ownership.
