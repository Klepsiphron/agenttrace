Install & run in under a minute

No accounts. No keys. No data leaves your machine.

1. Install the CLI (global)
bash 
npm install -g @agenttrace-io/cli

Or use npx agenttrace-io — no install needed.

2. Quick trace (TypeScript)
import { init } from '@agenttrace-io/sdk';

const agent = init({ dbPath: './agenttrace.db' });

const result = await agent.trace('search', async () => {
  return await myLLMCall(prompt);
}, { 
  input: prompt,
  tokens: { promptTokens: 120, completionTokens: 48, totalTokens: 168, model: 'gpt-4o' }
});

console.log(agent.getStats());
agent.close();
3. Explore with CLI
npx agenttrace-io init — create local DB
npx agenttrace-io dashboard — open web UI (http://localhost:4317)
npx agenttrace-io stats — summary numbers
npx agenttrace-io costs --daily — spend by day/model
npx agenttrace-io traces --limit 50 — recent calls
npx agenttrace-io tree --trace-id <id> — full agent tree
npx agenttrace-io self-stats — agent self-usage (today/week)
npx agenttrace-io export --format csv — or json / otel
Python SDK
bash 
pip install agenttrace-io

Decorator, context manager, and full parity with the TypeScript SDK.

Framework integrations: LangGraph • CrewAI • custom via SDK
